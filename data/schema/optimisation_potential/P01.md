## Beispiel: P01 - Teile-/Bauteilanzahl reduzieren

| Attribut         | Typ / Struktur                           | Befüllung                                                  | Beispielinhalt                                                                      |
| ---------------- | ---------------------------------------- | ---------------------------------------------------------- | ----------------------------------------------------------------------------------- |
| `potential_id`   | String                                   | Statisch / nicht durch LLM                                 | `P01`                                                                               |
| `potential_name` | String                                   | Statisch / nicht durch LLM                                 | `Teile-/Bauteilanzahl reduzieren`                                                   |
| `subpotentials`  | Liste von Subpotential Cards             | Statisch + LLM-Ergebnisse innerhalb der Subpotential Cards | `[SubpotentialCard(SP01_01), SubpotentialCard(SP01_02), SubpotentialCard(SP01_03)]` |
| `rating`         | Enum: `high` / `medium` / `low` / `none` | Berechnet / nicht durch LLM                                | Wird aus den enthaltenen Subpotential Cards berechnet                               |
