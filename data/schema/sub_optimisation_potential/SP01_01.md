## Beispiel: SP01_01 -  90°-Schweißverbindung durch Biegung ersetzen

| Attribut                        | Typ / Struktur                     |     Befüllung | Beschreibung                                                                                                                                                                                                                                                                                                                                                                            | Beispiel                                                                                                                                                                                            |
| ------------------------------- | ---------------------------------- | ------------: | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `subpotential_id`               | String                             |      Kein LLM | Eindeutige ID des Subpotentials                                                                                                                                                                                                                                                                                                                                                         | `TR-001`                                                                                                                                                                                            |
| `subpotential_name`             | String                             |      Kein LLM | 90°-Schweißverbindung durch Biegung ersetzen                                                                                                                                                                                                                                                                                                                                            | `90°-Schweißverbindung durch Biegung ersetzen`                                                                                                                                                      |
| `checkpoint_typ`                | Enum: `quantitativ` / `qualitativ` |      Kein LLM | `qualitativ`, weil die PDF keine Zahlen-Grenzwerte vorgibt, sondern eine konstruktive Erkennungssituation beschreibt                                                                                                                                                                                                                                                                    | `qualitativ`                                                                                                                                                                                        |
| `evaluation_goal`               | String                             | LLM-Bewertung | Bewerten, ob zwei separate Blechteile, die rechtwinklig miteinander verschweißt sind, zu einem einzigen Biegeteil zusammengeführt werden können, um die Teileanzahl zu reduzieren                                                                                                                                                                                                       | `Prüfen, ob Deckblech und Seitenblech einer Konsole als ein gebogenes Blechteil ausgeführt werden können.`                                                                                          |
| `check_method`                  | String                             | LLM-Bewertung | Suche im Bild/CAD nach separaten Blechteilen, die an einer rechtwinkligen, ca. 90°-Kante durch eine Schweißverbindung verbunden sind. Prüfe, ob diese Verbindung stattdessen als Biegelinie eines zusammenhängenden Blechteils darstellbar wäre. Nutze, falls vorhanden, die Abwicklung: Eine geeignete Lösung zeigt eine gemeinsame Kontur mit Biegelinie statt zwei getrennter Teile. | `Im CAD sind ein vertikales Seitenblech und ein horizontales Deckblech als getrennte Teile mit Schweißnaht an der 90°-Kante sichtbar.`                                                              |
| `potenzial_rating_logic.high`   | String                             | LLM-Bewertung | Hohes Potenzial, wenn mindestens eine klare 90°-Schweißverbindung zwischen separaten Blechteilen sichtbar ist und die beiden Flächen plausibel durch eine Biegung zu einem Teil zusammengeführt werden können.                                                                                                                                                                          | `Deckblech und Seitenblech sind separate Teile; die Naht liegt an einer klaren 90°-Kante; eine Biegelinie würde ein Teil einsparen.`                                                                |
| `potenzial_rating_logic.medium` | String                             | LLM-Bewertung | Mittleres Potenzial, wenn eine mögliche rechtwinklige Schweißverbindung erkennbar ist, aber aus Bild/CAD unsicher bleibt, ob es wirklich separate Teile sind, ob die Verbindung geschweißt ist oder ob die Abwicklung als ein Teil möglich ist.                                                                                                                                         | `Eine rechtwinklige Kontaktkante ist sichtbar, aber die Schweißnaht und die Teiletrennung sind im Bild nicht eindeutig erkennbar.`                                                                  |
| `potenzial_rating_logic.low`    | String                             | LLM-Bewertung | Geringes Potenzial, wenn keine rechtwinklige Schweißverbindung zwischen separaten Blechteilen erkennbar ist oder die Konstruktion bereits als ein gebogenes Einzelteil ausgeführt ist.                                                                                                                                                                                                  | `Die Konsole besteht bereits aus einem durchgehenden Biegeteil mit sichtbarer Biegelinie.`                                                                                                          |
| `rating`                        | Enum: `high` / `medium` / `low`    |  LLM-Ergebnis | Bewertung anhand der Rating-Logik                                                                                                                                                                                                                                                                                                                                                       | `high`                                                                                                                                                                                              |
| `reason`                        | String                             |  LLM-Ergebnis | Technische Begründung der Bewertung auf Basis beobachtbarer Hinweise                                                                                                                                                                                                                                                                                                                    | `Die Baugruppe enthält zwei getrennte Blechteile, die an einer 90°-Kante verschweißt sind. Diese Verbindung entspricht dem in der PDF gezeigten Fall und könnte durch eine Biegung ersetzt werden.` |
| `evidence`                      | String-Liste                       |  LLM-Ergebnis | Beobachtbare Hinweise aus Bild, CAD oder Abwicklung                                                                                                                                                                                                                                                                                                                                     | `["separates horizontales Deckblech", "separates vertikales Seitenblech", "sichtbare 90°-Kontaktkante", "Schweißnaht an der Kante", "mögliche Biegelinie in der Abwicklung"]`                       |
| `confidence`                    | Float `0.0–1.0`                    |  LLM-Ergebnis | Sicherheit abhängig von Sichtbarkeit der Teilegrenzen, Schweißnaht und Abwicklung                                                                                                                                                                                                                                                                                                       | `0.88`                                                                                                                                                                                              |
