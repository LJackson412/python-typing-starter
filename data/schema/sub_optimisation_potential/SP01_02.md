## Beispiel: SP01_02 — Sub-Potenzial: Geometrie ändern, um ein zusätzliches Teil zu integrieren

| Attribut                        | Typ / Struktur                     |     Befüllung | Beschreibung                                                                                                                                                                                                                                                                                                                    | Beispiel                                                                                                                                                                                                                        |
| ------------------------------- | ---------------------------------- | ------------: | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `subpotential_id`               | String                             |      Kein LLM | Eindeutige ID des Subpotentials                                                                                                                                                                                                                                                                                                 | `TR-002`                                                                                                                                                                                                                        |
| `subpotential_name`             | String                             |      Kein LLM | Geometrie ändern zur Integration eines zusätzlichen Blechteils                                                                                                                                                                                                                                                                  | `Geometrie ändern und Stützteil integrieren`                                                                                                                                                                                    |
| `checkpoint_typ`                | Enum: `quantitativ` / `qualitativ` |      Kein LLM | `qualitativ`, weil geprüft wird, ob eine konstruktive Lageänderung eines Bauteils eine Biegeteillösung ermöglicht                                                                                                                                                                                                               | `qualitativ`                                                                                                                                                                                                                    |
| `evaluation_goal`               | String                             | LLM-Bewertung | Bewerten, ob ein separates Zusatzteil, z. B. Unterstützungsstrebe oder Versteifung, durch eine einfache Geometrieänderung so positioniert werden kann, dass es als Teil eines gebogenen Einzelteils ausgeführt wird                                                                                                             | `Prüfen, ob eine angeschweißte Stützstrebe durch Verschieben an eine Biegekante in das Hauptblech integriert werden kann.`                                                                                                      |
| `check_method`                  | String                             | LLM-Bewertung | Suche nach separaten Zusatzteilen wie Streben, Laschen oder Versteifungen, die an ein Hauptblech angeschweißt sind. Prüfe, ob ihre Position oder Anschlussgeometrie so geändert werden könnte, dass die Schweißnaht durch eine Biegung ersetzt wird und das Zusatzteil in die Abwicklung des Hauptteils integriert werden kann. | `Eine diagonale Unterstützungsstrebe ist separat angeschweißt; durch Verschieben an den Rand des Hauptblechs könnte sie als gebogene Lasche integriert werden.`                                                                 |
| `potenzial_rating_logic.high`   | String                             | LLM-Bewertung | Hohes Potenzial, wenn ein separates Zusatzteil klar erkennbar ist, seine Funktion offenbar durch eine leicht geänderte Position erhalten bleiben kann und die Integration als Biegeteil plausibel ist.                                                                                                                          | `Die Strebe ist ein separates Blechteil ohne sichtbare Schnittstelle zu anderen Baugruppen; sie könnte an eine Biegelinie verschoben und in die Abwicklung aufgenommen werden.`                                                 |
| `potenzial_rating_logic.medium` | String                             | LLM-Bewertung | Mittleres Potenzial, wenn ein separates Zusatzteil erkennbar ist und eine Geometrieänderung möglich erscheint, aber funktionale Randbedingungen, Einbauraum, Lochpositionen oder Anschlussflächen nicht eindeutig bewertbar sind.                                                                                               | `Eine Strebe ist sichtbar, aber es ist unklar, ob ihre exakte Lage für Steifigkeit oder Montage zwingend erforderlich ist.`                                                                                                     |
| `potenzial_rating_logic.low`    | String                             | LLM-Bewertung | Geringes Potenzial, wenn kein separates Zusatzteil vorhanden ist, das durch Geometrieänderung integrierbar wäre, oder wenn die Lage des Zusatzteils offensichtlich durch Funktionsflächen, Bohrungen oder Bauraum festgelegt ist.                                                                                               | `Die Stützgeometrie ist bereits Bestandteil eines einteiligen Biegeteils oder die Strebe verbindet zwei funktional feste Anschlusspunkte.`                                                                                      |
| `rating`                        | Enum: `high` / `medium` / `low`    |  LLM-Ergebnis | Bewertung anhand der Rating-Logik                                                                                                                                                                                                                                                                                               | `medium`                                                                                                                                                                                                                        |
| `reason`                        | String                             |  LLM-Ergebnis | Technische Begründung der Bewertung auf Basis beobachtbarer Hinweise                                                                                                                                                                                                                                                            | `Eine separate diagonale Stützstrebe ist sichtbar. Eine Integration durch Geometrieänderung erscheint möglich, aber die genaue Funktionslage der Strebe und der verfügbare Bauraum sind aus dem Bild nicht vollständig sicher.` |
| `evidence`                      | String-Liste                       |  LLM-Ergebnis | Beobachtbare Hinweise aus Bild, CAD oder Abwicklung                                                                                                                                                                                                                                                                             | `["separate diagonale Stützstrebe", "Schweißkontakt zum Hauptblech", "Hauptblech mit potenzieller Biegekante", "keine eindeutige Abwicklung für integrierte Strebe sichtbar"]`                                                  |
| `confidence`                    | Float `0.0–1.0`                    |  LLM-Ergebnis | Sicherheit abhängig von Sichtbarkeit der Strebenfunktion, Teilegrenzen, Schweißnaht und Abwicklung                                                                                                                                                                                                                              | `0.72`                                                                                                                                                                                                                          |
